Practica General RA3.7

Clima RD Hoy

Evaluacion de calidad de un sitio web

Esmaylin Jimenez Rosa

Objetivo

Consultar el clima actual de Santo Domingo

El sitio usa una API externa para mostrar temperatura, viento y una alerta del clima. Tambien permite suscribirse a notificaciones simuladas.

Tecnologias

Herramientas utilizadas

  • HTML para la estructura.
  • CSS para el diseno visual y responsivo.
  • JavaScript para la interactividad.
  • Open-Meteo para datos del clima.
  • Selenium basico para pruebas automatizadas.

Funcionalidades

Que permite hacer el sitio

  • Consultar temperatura y viento.
  • Ver el estado de conexion de la API.
  • Clasificar el clima como muy caluroso, caluroso, normal, algo frio o frio.
  • Suscribirse y cancelar la suscripcion.

Demostracion

Integracion con API

La seccion API Clima muestra datos reales obtenidos desde Open-Meteo. Si la API responde correctamente, aparece el estado “Conectada”.

Mostrar en vivo: temperatura, viento y alerta actual.

Demostracion

Suscripcion a alertas

El usuario escribe su nombre y correo, presiona Suscribirme y el sistema guarda la informacion en el navegador.

Mostrar en vivo: tarjeta de suscripcion activa.

Demostracion

Cancelar suscripcion

El boton Cancelar suscripcion elimina los datos guardados y oculta la tarjeta de usuario suscrito.

Mostrar en vivo: mensaje de cancelacion.

Pruebas

Escenarios realizados

  • Prueba funcional.
  • Prueba responsiva.
  • Prueba de API.
  • Validacion en Google Chrome y Mozilla Firefox.

Errores

Hallazgos documentados

  • Desplazamiento horizontal en vista movil.
  • Solicitud intencional a JSON inexistente.
  • Suscripcion local sin envio real de correos por falta de backend.

Conclusion

Resultado final

Clima RD Hoy cumple con los requisitos de la practica: sitio creado en HTML, CSS y JavaScript, integracion con API, pruebas, guia de uso, informe y lista de cotejo.
